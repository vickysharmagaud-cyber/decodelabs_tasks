# Data Cleaning & Preparation

## Project Overview

This project was completed as part of the DecodeLabs Data Analytics Internship.

The objective of this project is to clean and prepare an e-commerce dataset using Python and Pandas.

---

## Objectives

- Handle missing values
- Remove duplicate records
- Convert data types
- Verify data consistency
- Export cleaned dataset

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Data Cleaning Steps

1. Imported the dataset
2. Checked missing values
3. Filled missing CouponCode values with "No Coupon"
4. Checked duplicate records
5. Converted Date column to datetime format
6. Verified TotalPrice calculation
7. Exported cleaned dataset

---

## Results

- Missing values handled successfully
- Date column converted to datetime
- Dataset cleaned and validated
- Cleaned dataset exported successfully
